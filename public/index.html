Totally Normal Text Editor is a text editor with a secret: type any word, then press space or Enter, and it might come alive.

Behind the scenes, an AI reads the word's meaning and writes a small piece of code describing how it should move. The first time anyone types a new word, that code gets generated and added to the collection, so it takes a moment to appear. After that, the word animates instantly for everyone.

Most words stay perfectly still. A few don't.
